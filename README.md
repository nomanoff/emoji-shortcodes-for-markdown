
# Over [1000+ Emoji Emoticons](https://emoji-ninja.netlify.app/) For Markdown, GitHub, Slack and more

#### We all love using emoji emoticons, don't we? :smile:
#### However, some of them are not super easy to remember when needed.  :confused:
#### This is where   **[emoji-ninja.netlify.app](https://emoji-ninja.netlify.app/)** comes into play!

<a  href="https://emoji-ninja.netlify.app/"><img  width="728"  src="/src/img/emojicodes.png"  alt="Demo"></a>

> Search for your favorite emoji and copy it with just ONE CLICK
  

## Why

-  ✔ Searchable  
-  ✔ Easy Access 
-  ✔ Click to Copy
-  ✔ 1000+ Emoticons
-  ✔ Simple & Modern UI  


  
## Usage

>Hover over and CLICK to Copy to Your Clipboard
